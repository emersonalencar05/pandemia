GIT & GITHUB

Git

Git é um sistema de controle de versões, bastante usado por desenvolvedores para desenvolvimento de software, registros de históricos e edição de qualquer tipo de arquivo.

O Git é prático, pois armazena históricos dos projetos criados em suas versões. Também pode ser um repositório de um grupo onde todos possam modificar o projeto sem perdê-lo. Por isso o Git e uma ferramenta bastante utilizada por empresas de grande porte como: Facebook, Adobe Systems, Google entre outras.

Se não houvesse um controle de versão, imagine o caos entre duas pessoas abrindo um arquivo ao mesmo tempo. Uma das ferramentas do Git e justamente essa, permitir que um arquivo possa ser editado ao mesmo tempo, por diferentes pessoas. Por mais "bagunça" que isso faça, o Git tenta deixar tudo isso bem organizado, para evitar problemas para os desenvolvedores do projeto.

Outro fato do Git e poder criar, a qualquer momento, vários snapshots do seu trabalho. Algumas ferramentas do Git:

  • Aliases (usado como um atalho, para agilizar comandos);
  • Hooks (são comandos que são executados de acordo com certas ações do Git);
entre outras.

O Git foi inicialmente programado e desenvolvido por Linus Torvalds para o desenvolvimento do Kernel Linux, porém sofreu adaptações.

Github

A ferramenta do Git realmente e ótima para a construção de projetos, mas ai esta um problema! Como os outros participantes do grupo poderiam ter acesso a todo esse código e adicionar também seus branchs e commits? Então criamos a "rede social" dos programadores o Github! Git e Github são bem diferentes apesar de eles se complementarem! O git como dito e uma ferramenta de modificação, que auxilia o programador! Já o Github é um serviço de hospedagem de arquivos Git, repositórios de Git (como e chamado). Ele pode manter todos os branchs e commits de todos os membros atualizados e sincronizados entre o time.

O Github funciona como um compartilhamento e hospedagem de dados. O Github possui endereço para dispositivos moveis, como o seu aplicativo o OpenHub, e também possui sua página na Web. Você entra no github.com/, cria seu perfil e pode aproveitar das ferramentas do Github! No Github você pode ver projetos de outros usuários, seguir-los e compartilhar os seus trabalhos com quem você quiser!

O Git e o Github, contribuem muito para a criação de projetos no dia a dia dos programadores. São ferramentas que se complementam para da (suporte) ao usuário!
